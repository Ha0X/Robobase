## Ch02 Configuration Space（构型空间 / 位形空间）

> 本章将机器人建模为刚体连杆组成的系统。

### 1) 核心问题：机器人“在哪里”？

对于机器人来说，首先要关注的是它在哪里、如何描述它的“状态”。  
**机器人位形（configuration）**是机器人上**每个点的位置的完整描述**。

但由于连杆是**刚性**且形状已知，很多系统只需要少量坐标就能表示整体位形：

- 门的位形：可以用铰链角度 $\theta$ 描述
- 平面点：可以用 $(x,y)$ 描述

### 2) 自由度（DOF）与 C-space 的定义

机器人的自由度是表示其位形所需的最小实值坐标数。



- **C-space（构型空间）**：所有可能位形组成的空间
- **一个位形**：C-space 里的一个点

### 3) DOF 怎么算：自由度 = 自由变量 − 独立约束

一个通用的思路是：

$$\text{dof} = (\text{自由变量数量}) - (\text{独立约束数量})$$

如果是机构/机器人这种“连杆 + 关节”系统，常见的做法是用 **Grübler–Kutzbach 公式**来估算自由度。



**平面机构常用形式**：

$$M = 3(L-1) - 2J_1 - J_2$$

- $L$：连杆数（含机架）
- $J_1$：1 自由度低副数（转动/移动副）
- $J_2$：2 自由度高副数

### 4) 任务空间与工作空间（Task Space & Workspace）

关注完 C-space 后，本章还引入与末端执行器相关的两个概念：**任务空间**和**工作空间**。  
它们都只关心末端执行器（end-effector），而不是整个机器人的位形。

#### 4.1 任务空间（Task Space）

原书定义：

> The task space is a space in which the robot’s task can be naturally expressed.

例子：

- 如果任务是用笔在纸上绘图：任务空间可自然选为二维平面（类似 $\mathbb{R}^2$）
- 如果任务是操纵刚体：任务空间常用刚体位姿（位置 + 方向），即末端坐标系在空间中的位姿

关键点：**任务空间怎么定义由任务决定，与机器人本身无关**。

#### 4.2 工作空间（Workspace）

原书定义：

> The workspace is a specification of the configurations that the end-effector of the robot can reach.

关键点：

- 工作空间主要由**机器人结构**决定（连杆长度、关节限制、是否有碰撞等）
- 与“任务怎么定义”无关
